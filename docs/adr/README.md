# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the
OpenCode Autonomous Team project. ADRs capture the context, rationale, and
consequences of significant architectural decisions.

## Format

Records follow the format defined in
[ADRF-0001](./ADRF-0001-use-adrs.md):

```markdown
# ADRF-NNNN: Title
**Status:** proposed | accepted | deprecated | superseded
**Date:** YYYY-MM-DD

## Context
## Decision
## Consequences
```

## Status

- **proposed** -- Under discussion, not yet adopted.
- **accepted** -- Adopted and in force.
- **deprecated** -- No longer recommended but still in use.
- **superseded** -- Replaced by a newer ADR.
