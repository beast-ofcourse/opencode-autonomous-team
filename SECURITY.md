# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by opening a GitHub issue. Do not disclose the issue publicly until we have had a chance to address it.

We will acknowledge receipt of your report within 48 hours and work with you to understand and resolve the issue.

## Response Timeline

- **Acknowledgment**: Within 48 hours of reporting
- **Assessment**: We will triage and determine severity
- **Fix**: A timeline for the fix will be communicated based on severity
- **Disclosure**: We will coordinate public disclosure with you after the fix is released

## Supported Versions

We currently support the latest release with security updates. If you are using an older version, please upgrade to the latest release.
